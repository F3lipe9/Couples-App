# Quick Start Guide

## Start the Backend Server (FastAPI)
```bash
cd backend
python main.py
```
Backend runs on: http://localhost:5000
API docs available at: http://localhost:5000/docs

## Start the Frontend (in another terminal)
```bash
npm run dev
```
Frontend runs on: http://localhost:5173

## First Time Setup
1. Backend is already configured with your MongoDB Atlas connection
2. Python dependencies are already installed
3. Make sure both servers are running before using the app
4. Sign up with any email/password to create your account

## API Documentation
FastAPI provides automatic interactive API docs at:
- Swagger UI: http://localhost:5000/docs
- ReDoc: http://localhost:5000/redoc
